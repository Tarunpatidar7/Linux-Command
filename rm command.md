# rm command 
* rm  :- rm file delete
## rm remove the file
```
rm filename 
```
## rm munltiple file delete
```
rm filename1 filename2 
```
## rm force confirmation then delete
```
rm -f filename
```
## rm confirmation delete command 
``` 
rm -v filename
```
## rm recursively delete command 
```
rm -r filename
```
## prompt confirmation delete
```
rm -i filename 
```
## only directories delete
```
rm -d foldername 
```
## --help :- help command
```
rm --help
```
